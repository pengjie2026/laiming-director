# 籁鸣导演 — 儿童AIGC动画创作平台
## 产品方案及设计文档 v1.0

> **让每个孩子的童话梦想，都能成为专业级动画作品**

---

## 一、产品概述

### 1.1 产品定位

**籁鸣导演**是专注于儿童动画创作领域的AIGC垂直服务平台，对标「即梦」「liblib」「oiioii」在儿童动画领域的垂直细分应用。

**核心差异化**：
- 面向**专业动画创作者**，不是低幼儿童自主使用
- **AI生成内容为核心**，其他功能为辅助
- 内置**7大专业AI Agent**协同工作
- **儿童内容专属**：广电标准审核、教育价值提炼

### 1.2 产品愿景

成为儿童动画创作者最信赖的AI创作伙伴，让专业动画制作触手可及。

### 1.3 核心价值主张

| 价值 | 描述 |
|------|------|
| 全流程AI赋能 | 剧本→分镜→生图→视频→剪辑→配音 |
| 7大专业Agent | 艺术总监/编剧/角色设计/分镜/场景/剪辑/音效 |
| 角色一致性 | 跨镜头保持角色形象统一 |
| 广电标准审核 | 内置少儿节目制作规范 |
| 教育价值提炼 | 自动提炼教育属性点 |

## 二、7大核心专业AI Agent

### Agent 1：艺术总监
- 统筹整体视觉风格，把控创作方向
- 风格定义、色彩管理、视觉统筹

### Agent 2：资深编剧
- 专业儿童向剧本创作，符合广电标准
- 剧本创作、故事结构、对白优化

### Agent 3：角色设计师（核心）
- **跨镜头角色一致性控制**（对标OiiOii的DreamActor-M1）
- 角色设定、表情系统、一致性锁定

### Agent 4：分镜师
- 专业动画分镜设计，镜头语言精准
- 分镜设计、镜头语言、节奏控制

### Agent 5：场景创作者
- 儿童动画专属场景设计，温暖安全
- 场景生成、背景设计、道具管理

### Agent 6：剪辑师
- 智能视频剪辑合成，节奏自动匹配
- 自动剪辑、智能转场、字幕生成

### Agent 7：音效总监
- 儿童向配音配乐音效全套方案
- 配音指导、背景音乐、音效设计

## 三、核心功能模块

### 3.1 智能剧本创作系统

**功能特性**：
- 创意构思辅助：输入关键词，AI扩展为完整故事大纲
- 专业剧本撰写：标准动画剧本格式，符合行业规范
- 剧本优化建议：结构/节奏/对白专项优化
- 教育价值提炼：自动分析并提炼友谊/勇气等教育属性

**AI模型支持**：MiniMax / GPT-4 / Claude

### 3.2 分镜设计系统

**功能特性**：
- 剧本自动分镜：一键转换为带镜头描述的分镜脚本
- 镜头语言建议：推/拉/摇/特写等专业镜头指导
- 分镜参考图生成：单张或九宫格参考画面
- 时间轴规划：智能规划场景时长

### 3.3 视觉生成系统（对接MiniMax）

**风格库（140+）**：

| 画风 | 适龄 | 参考作品 |
|------|------|---------|
| Q版卡通 | 3-6岁 | 小猪佩奇、汪汪队 |
| 绘本风格 | 3-8岁 | 猜猜我有多爱你 |
| 日式清新 | 6-12岁 | 宫崎骏系列 |
| 3D立体 | 全年龄 | 冰雪奇缘、寻梦 |
| 国风古韵 | 6-12岁 | 中国唱诗班、哪吒 |
| 粘土定格 | 3-10岁 | 小羊肖恩 |

**输出格式**：单张 / 四宫格 / 九宫格

### 3.4 动画视频生成系统（对接MiniMax）

**生成模式**：
- 文生视频：基于分镜描述直接生成动画片段
- 图生视频：基于参考图片生成动态效果

**关键参数**：
- 时长：3s / 5s / 10s
- 比例：9:16竖屏 / 16:9横屏 / 1:1
- 运镜：固定/推镜/拉镜/摇镜

**儿童动画专属规范**：
- 动作夸张有趣（吸引儿童注意）
- 色彩鲜艳明快
- 避免恐怖/阴暗元素

### 3.5 资产管理系统

- 项目库管理：每个项目独立空间
- 角色资产库：形象/表情/动作资产统一管理
- 场景资产库：背景/道具/环境分类管理
- 跨项目复用：资产导入其他项目

### 3.6 儿童内容安全审核（差异化核心）

**审核标准**：
- 《未成年人节目管理规定》
- 《少儿节目制作规范》
- 广电总局相关标准

**审核维度**：

| 维度 | 检测内容 |
|------|---------|
| 暴力/恐怖 | 打斗、伤害、黑暗、惊吓 |
| 价值观 | 歧视、攀比、不当引导 |
| 语言规范 | 粗俗用语、不当表达 |
| 广告合规 | 过度营销、诱导消费 |
| 适龄性 | 内容复杂度、情感表达 |

**输出功能**：
- 合规评分（0-100分）
- 具体问题定位
- AI修改建议
- 导出合规报告（用于送审备案）

## 四、儿童动画专业工作流规范

### 4.1 创作工作流（6步）

```
Step 1：剧本创作
  └── 故事概念 → 大纲 → 分集大纲 → 完整剧本
  └── AI辅助：构思发散、结构优化、对白改写

Step 2：分镜设计
  └── 剧本分镜化 → 镜头语言设计 → 时间轴规划
  └── AI辅助：自动分镜、镜头建议

Step 3：视觉风格确定
  └── 风格选择 → 角色设计 → 场景设计 → 资产固定
  └── 参数：年龄段 × 性别向 × 画风类型

Step 4：动画制作
  └── 逐场生成视频片段 → 镜头合成
  └── 方式：文生视频 / 图生视频

Step 5：剪辑合成
  └── 智能剪辑 → 转场 → 字幕 → 音效
  └── 参考：每15-30秒一个音效点

Step 6：配音音效
  └── AI配音 → 背景音乐选配 → 音效设计
  └── 儿童动画音效规范：夸张、即时、避免恐怖音
```

### 4.2 配音配乐规范（来源：老棚的知识库）

**最佳配乐介入时机**：
- 场景设置确定后
- 画风风格确定后
- 能提供比较成形的场景图之后

**儿童动画音效特殊规范**：
- 音效要**夸张**：比真实声音更响亮、更有趣
- 音效要**即时**：与画面动作精确同步
- 避免**恐怖音效**
- 多用**卡通音效**：弹簧声、滑音、卡通撞击声

**情绪与音乐对应**：

| 情绪/场景 | 音乐特征 | 乐器 |
|---------|---------|------|
| 欢快/儿童 | 明快、节奏轻盈 | 木琴、钢琴 |
| 温馨/家庭 | 平缓、温暖 | 钢琴、吉他 |
| 冒险/热血 | 节奏强劲 | 铜管、打击乐 |
| 搞笑/反转 | 短促、错位 | 卡通音效 |

### 4.3 分镜叙事规范

**儿童动画经典叙事结构**：
```
A结构（3-6岁适用）：
  建立情景 → 遇到小问题 → 解决问题 → 快乐结局

B结构（7-12岁适用）：
  角色目标 → 障碍/挑战 → 伙伴帮助/自我成长 → 实现目标 → 收获感悟
```

**镜头语言规范**：
- 3-6岁：多用全景/中景，减少快速剪辑，特写要柔和
- 7-12岁：可适当使用多样化镜头，加入悬念设置

## 九、产品经理视角补充建议

1. **版权与IP保护**：为创作者生成内容提供版权登记服务，与版权局合作
2. **创作者社区**：类似liblib的社区功能，展示优质儿童动画作品
3. **教育机构合作**：与幼儿园/小学/培训机构合作，提供教育版
4. **内容分发渠道**：接入优酷少儿、爱奇艺奇巴布、抖音儿童等平台
5. **AI模型持续迭代**：针对儿童动画风格进行LoRA微调，提升专业度
6. **行业标准制定**：参与儿童AIGC内容标准制定，建立行业话语权
7. **离线能力**：部分功能支持离线使用，适应教育场景
8. **多语言支持**：中英日等，拓展国际儿童动画市场
